This folder contains the evaluation format - expanded with coreference relations from ECB+ - for plot link relations 
as obtained from crowd workers.
Original crowd data: https://github.com/CrowdTruth/Crowdsourcing-StoryLines
Paper: 
* Tommaso Caselli and Oana Inel: [Crowdsourcing StoryLines: Harnessing the Crowd for Causal Relation Annotation](https://www.aclweb.org/anthology/W18-4306.pdf). [Events and Stories in the News Workshops, COLING 2018](http://www.eventstory.news)
